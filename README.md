# Responsive Signup and Signin Page

A responsive web application featuring Signup and Sign-In pages with real-time form validation.

## Features
* **Responsive Design:** Works smoothly on mobile, tablet, and desktop screens.
* **Form Validation:** Client-side validation for emails, passwords, and phone numbers using JavaScript.
* **Password Toggle:** Option to view or hide the password while typing.
* **Local Storage:** Simulates user authentication by storing and verifying credentials locally in the browser.

## Technologies Used
* HTML5
* CSS3 & Bootstrap 5
* Vanilla JavaScript

## Project Structure
* `SignUp.html` - Registration page
* `SignIn.html` - Login page
* `travelapp.html` - Landing page after successful login
* `css/style.css` - Custom styles
* `js/index.js` - Validation and authentication logic

## How to Run
1. Clone this repository to your local machine.
2. Open `SignUp.html` in any modern web browser to start the application.
